【设计模式 & 关键特性】
============================================================

  ✓ Provider 抽象层      — 支持多 LLM 后端切换
  ✓ 工具注册表           — 动态发现 / OpenAPI schema 构建
  ✓ 插件系统             — HookManager 支持钩子注入
  ✓ 上下文压缩           — 自动摘要防止 token 溢出
  ✓ 审批流程             — 高风险工具需用户确认
  ✓ 护栏机制             — 执行前安全检查
  ✓ 流式输出             — 实时打印 + tool_calls 拼接
  ✓ 会话持久化           — 保存/恢复，自动生成标题
  ✓ 定时提醒             — Cron 调度器集成
  ✓ Token 用量追踪       — UsageTracker 统计成本
  ✓ 长期记忆             — MEMORY.md 注入 system prompt
  ✓ 重试机制             — retry_call 包装 API 调用
  ✓ MCP 集成             — 多服务器连接
  ✓ 技能系统             — 可复用技能 + 自动 Nudge
  ✓ 自升级提示           — 复杂任务后建议创建技能
