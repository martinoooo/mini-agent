# Mini Agent 开发路线图

## 已完成

- [x] **Step 1** — Hello LLM：最简单的 API 调用，理解 system/user/assistant 三个角色
- [x] **Step 2** — 第一个工具：Function Calling 机制，LLM 决定是否调用工具
- [x] **Step 3** — 工具注册表：自注册模式，加工具不改核心代码
- [x] **Step 4** — 工具集系统：按场景组合工具，递归展开 includes
- [x] **Step 5** — Agent 循环：AIAgent 类，LLM → 工具 → LLM 完整闭环
- [x] **Step 6** — Web 搜索：扩展验证自注册模式（加工具只需新建一个文件）
- [x] **Step 7** — 流式输出：stream=True，逐 token 实时打印，理解 SSE 协议
- [x] **Step 8** — 会话持久化：JSON 文件存储，保存/恢复对话历史
- [x] **Step 9** — 上下文压缩：token 窗口管理，长对话自动摘要

## 待开发

### 工具能力

- [ ] **浏览器工具** — Playwright 集成，页面导航/截图/点击，理解异步工具编排
- [ ] **代码执行沙箱** — Docker 隔离执行，安全运行用户代码
- [ ] **MCP 协议集成** — 连接 MCP Server，接入外部工具生态
- [ ] **图像生成** — 接入 DALL-E/Stable Diffusion，理解多模态输出
- [ ] **语音交互** — TTS/STT，语音输入输出

### 智能体能力

- [ ] **多模型适配** — 支持 OpenAI / Anthropic / Gemini 等多提供商切换
- [ ] **子 Agent 委派** — Agent spawn 子 Agent 做子任务，并发控制
- [ ] **凭证池** — 多 API Key 负载均衡和故障转移
- [ ] **记忆系统** — 文件型记忆（MEMORY.md），自动读写关键信息，RAG 雏形
- [ ] **技能系统** — 从对话中学习，自动创建可复用的技能文档
- [ ] **会话搜索** — 全文搜索历史对话，跨会话知识检索

### 安全与可靠性

- [ ] **命令审批** — 危险命令需要用户确认后才执行
- [ ] **工具护栏** — 检测异常行为（无限循环、敏感文件访问等）
- [ ] **错误分类与重试** — API 错误自动分类，智能重试策略
- [ ] **PII 脱敏** — 自动检测和脱敏敏感信息

### 交互与体验

- [ ] **Web 仪表板** — 浏览器中的管理面板，可视化会话和配置
- [ ] **消息平台接入** — Telegram / Discord / 飞书 / 企业微信等
- [ ] **TUI 高级界面** — 基于 Rich/Textual 的终端 UI
- [ ] **插件系统** — Python 插件 + Shell Hook，可扩展生命周期钩子

### 工程化

- [ ] **配置系统** — YAML 配置文件，统一管理所有参数
- [ ] **日志系统** — 结构化日志，分级输出
- [ ] **测试套件** — 单元测试 + 集成测试，回归保护
- [ ] **使用统计** — token 用量、成本估算、调用频率分析
- [ ] **定时任务** — Cron 调度，Agent 自主管理定时提醒
