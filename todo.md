# Mini Agent 开发路线图

## 已完成

- [x] **Step 1** — Hello LLM：最简单的 API 调用，理解 system/user/assistant 三个角色
- [x] **Step 2** — 第一个工具：Function Calling 机制，LLM 决定是否调用工具
- [x] **Step 3** — 工具注册表：自注册模式，加工具不改核心代码
- [x] **Step 4** — 工具集系统：按场景组合工具，递归展开 includes
- [x] **Step 5** — Agent 循环：AIAgent 类，LLM → 工具 → LLM 完整闭环
- [x] **Step 6** — Web 搜索：扩展验证自注册模式（加工具只需新建一个文件）
- [x] **Step 7** — 流式输出：stream=True，逐 token 实时打印，理解 SSE 协议
- [x] **Step 8** — 会话持久化：JSON 文件存储，保存/恢复对话历史
- [x] **Step 9** — 上下文压缩：token 窗口管理，长对话自动摘要
- [x] **Step 10** — 代码执行沙箱：subprocess 隔离，超时控制，输出截断
- [x] **Step 11** — 命令审批：风险分级，交互式 y/n 确认，回调注入
- [x] **Step 12** — 记忆系统：MEMORY.md 自动读写，启动时注入上下文
- [x] **Step 13** — 工具护栏：循环检测、敏感路径拦截、超长命令检测
- [x] **Step 14** — 错误分类与重试：指数退避，可重试 vs 不可重试
- [x] **Step 15** — 多模型适配：Provider 抽象层，适配器模式
- [x] **Step 16** — 配置系统：JSON 配置文件，环境变量覆盖
- [x] **Step 17** — 日志系统：双输出（控制台+文件），分级日志
- [x] **Step 18** — 定时任务：Cron 调度，后台线程轮询，到期提醒注入
- [x] **Step 19** — 使用统计：token 追踪、工具计数、成本可视化
- [x] **Step 20** — 子 Agent 委派：任务隔离，独立消息历史，结果聚合
- [x] **Step 21** — 会话搜索：全文搜索历史对话，跨会话知识检索
- [x] **Step 22** — 技能系统：从对话中学习，创建可复用技能文档
- [x] **Step 23** — PII 脱敏：敏感信息自动检测和隐藏
- [x] **Step 24** — 凭证池：多 API Key 轮询，故障转移
- [x] **Step 25** — 测试套件：29 个单元测试，核心模块回归保护
